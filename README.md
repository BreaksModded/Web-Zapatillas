# Web de Servicios de Limpieza de Zapatillas

Este proyecto es una página web para un servicio artesanal de limpieza de zapatillas. La web está diseñada para ser moderna, atractiva y con animaciones interactivas.

## Características

- Diseño moderno y responsive
- Animaciones y efectos interactivos
- Secciones para mostrar servicios, proceso de limpieza, galería y testimonios
- Formulario de contacto funcional
- Optimizado para dispositivos móviles y de escritorio

## Estructura del Proyecto

- `index.html` - Estructura principal de la página web
- `styles.css` - Estilos y diseño visual
- `script.js` - Funcionalidades interactivas y animaciones

## Cómo Usar

1. Abre el archivo `index.html` en tu navegador para ver la página web.
2. Personaliza el contenido según tus necesidades:
   - Actualiza los textos en `index.html`
   - Modifica los colores y estilos en `styles.css`
   - Ajusta las animaciones en `script.js`

## Personalización

### Imágenes

Actualmente, la web utiliza imágenes de placeholder. Para un sitio web real, deberías:

1. Reemplazar la imagen de fondo del hero en `styles.css` (busca la clase `.hero`)
2. Añadir imágenes reales de antes/después en la sección de galería
3. Actualizar los iconos o añadir un logo personalizado

### Información de Contacto

Actualiza la siguiente información en `index.html`:

- Dirección física
- Número de teléfono
- Correo electrónico
- Enlaces a redes sociales

### Colores

El esquema de colores principal se puede modificar en `styles.css`. Busca los gradientes y colores principales:

```css
background: linear-gradient(90deg, #3498db, #1abc9c);
```

## Tecnologías Utilizadas

- HTML5
- CSS3 (con Flexbox y Grid para layouts)
- JavaScript (vanilla, sin frameworks)
- Font Awesome para iconos
- Google Fonts para tipografías
